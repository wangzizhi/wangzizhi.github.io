Css选择器基础

选择器主要分为以下几类

  • 元素选择器
  • 类选择器
  • Id选择器
  • 子类选择器
  • 属性选择器
  • 兄弟选择器
  • 伪类和伪元素选择器

前面4类选择器使我们常用到的，相较而言与我们比较熟悉，这里我们变不再做过多赘述了。

后面3类选择器我们不常用到，但其实妙用大多来自这3类选择器中，尤其是伪类和伪元素选择器提供了相当强大的功能。下面就让我们来了解一下。

1.属性选择器 [ ]

属性选择器是基于元素选择器、类选择器和Id选择器之上的，也就是说你不能直接定位到有这个属性的元素，而是需要指定哪一类文档元素有或者没有该属性。 代码示例

3 伪类和伪元素选择器 //!important

这里要着重区别下 伪类和伪元素

伪类：类似通常HTML中的类，但是它们不会直接以html标记的形式定义，而是一个动态的，满足一定条件时才会显示出来特定的效果。

伪元素：主要这里的用词是元素，虽然是不存在于dom树种，但其本质还是一种动态元素。

伪类总集

例子 类别 解释
a:link 链接(link)伪类 选择一个用户没有访问过的链接

p:last-of-type 结构性伪类 选择父元素中同类型的最后一个子元素
img:only-of-type 结构性伪类 若元素为父元素中同类型的唯一元素，则会被匹配
li:nth-child(2n+3) 结构性伪类 选择以第3个子元素为开始，2为等差数向后递增的子元素中，如果该子元素为li元素则样式生效，不是li元素样式不会生效。即选中 3 5 7 9 11 … 中是li元素的子元素赋予样式
li:nth-last-child(3n+2) 结构性伪类 选择以倒数第二个子元素为开始，3为等差数向前递减的子元素中，如果该子元素为li元素样式生效，不是li元素样式不会生效
p:nth-of-type(3n) 结构性伪类 选择父元素下的所有p元素提取出来，以0为开始，3为等差数向后递增的p元素，对子元素的排序没有要求,即俩个p元素之间可以插入任意多个元素，不会影响最终效果
p:nth-last-of-type(2n+1) 结构性伪类 选择父元素下的所有p元素提取出来，以倒数第一个p元素为开始，2为等差数向前递减的p元素，对子元素的排序没有要求，即俩个p元素之间可以插入任意多个元素，不会影响最终效果
section:target 目标伪类 选择URI片段标示符的值指向的元素
div:empty 空伪类 选择没有任何子元素或者文本的元素
div:not(.awesome) 否定伪类 选择不匹配某个状态标示符的元素

总结下

  1. 所有伪类中带child的，针对的是父元素下的所有子元素集进行排序,而所有伪类中带type的，针对的是父元素下的指定元素集进行排序
  2. (2n+0)的表达式中 n旁边的数字代表等差数，+号左或者右的数字代表起始计算位置。
  3. 所有伪类选择的都是一个元素集，并不是单一元素，直选中单一元素只是应为你只有一个元素是满足选择条件的。

个人感悟

其实一个页面的结构一定程度上影响了写css时，对选择器的选择。建议先整体构思页面结构，找出如何才能更少，更简单地使用的css的选择器就能实现你想要的页面效果。
